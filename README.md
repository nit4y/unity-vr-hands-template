# Unity VR project template with hands
Unity VR project with hand entities that you can use as a base for a VR project.

### Features
1. Hand entities
2. Basic box grabbable
3. Basic Locomotion
4. Universal RP integration

### Tested headsets
Rift S

### Tested Unity Versions
2021.3.20f1
2021.3.11f1
